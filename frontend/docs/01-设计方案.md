## 一、设计理念与品牌基调

### 1.1 核心设计理念
- **稳重大气**：贴合国资国企严肃、权威的品牌属性
- **专业严谨**：中后台工业级设计语言，信息密度高、层级清晰
- **AI原生**：每个页面预留AI助手入口、流式输出体验、Agent工作流可视化
- **数据驱动**：深度融合ECharts可视化，所有决策界面配可视化
- **中国风点缀**：金色线条、印章式徽章、祥云纹理点缀，体现国资文化底色

### 1.2 设计关键词
> 大气 · 红色 · 深蓝 · 金色 · 庄重 · 现代 · 清晰 · 高效

## 三、字体规范

### 3.1 字族
- **中文**：Microsoft YaHei / PingFang SC / Source Han Sans CN
- **中文点缀**：SimSun / STSong（用于Banner大标题的庄重感）
- **数字与英文**：Inter / SF Pro Display / Helvetica Neue
- **代码**：JetBrains Mono

### 3.2 字号层级
| 级别 | 字号 | 字重 | 用途 |
|------|------|------|------|
| Display | 72px | 800 | 信用评分巨幅数字 |
| H1 | 32px | 700 | 页面主标题、Banner标语 |
| H2 | 20px | 700 | 卡片标题、分区标题 |
| H3 | 15px | 700 | 子卡片标题 |
| Body | 14px | 400 | 正文 |
| Small | 12-13px | 400 | 辅助信息 |
| Caption | 11px | 400 | 标签、图例 |

### 3.3 数字字号加重
所有KPI数字、评分、金额统一使用 `font-family: Inter, monospace`，且 `font-variant-numeric: tabular-nums` 等宽排版，视觉更专业。

## 四、布局与栅格

### 4.1 基础栅格
- 页面最小宽度：`1280px`
- 优化分辨率：`1920×1080`（PC大屏首要目标）
- 栅格系统：12栅格（Tailwind Grid）
- 统一内边距：`32px`（页面左右）、`24px`（卡片内）
- 模块间距：`16-24px`

### 4.2 页面结构模板
```
┌─────────────────────────────────────────────┐
│  顶部品牌条（深红，40px高）                   │
├─────────────────────────────────────────────┤
│  主导航（深蓝，50px高，七智能体+驾驶舱）       │
├─────────────────────────────────────────────┤
│  [面包屑  或  Banner]                         │
├───────────┬─────────────────────────────────┤
│  左侧导航  │                                 │
│  (220px)  │     主内容区                     │
│           │                                 │
├───────────┴─────────────────────────────────┤
│  页脚（深蓝）                                 │
└─────────────────────────────────────────────┘
```

### 4.3 响应式断点
| 断点 | 宽度 | 策略 |
|------|------|------|
| xl+ | ≥1600px | 完整布局，4列智能体卡片 |
| lg | 1280-1599px | 字号缩小0.95x |
| md | 1024-1279px | 不作重点适配（中后台定位） |

## 五、组件规范

### 5.1 卡片（Card）
- 圆角：`6px`（小圆角，保持稳重）
- 边框：`1px solid #E5E1D6`
- 阴影：`0 1px 2px rgba(15,42,86,0.06)`
- Hover：`0 12px 32px rgba(184,20,26,0.12)` + `translateY(-2px)`
- 标题栏左侧：金红渐变竖条 `3×16px`

### 5.2 按钮（Button）
| 类型 | 样式 | 用途 |
|------|------|------|
| Primary | 红色渐变背景 `linear-gradient(135deg, #B8141A, #8A0F14)` | 主要操作 |
| Secondary | 白底 + 深蓝边框与文字 | 次要操作 |
| Gold | 金色渐变背景 | 特殊/重要操作（AI解读、驾驶舱入口等） |
| Ghost | 透明 + 浅边框 | 中性操作 |
| 尺寸 | 默认 `8×18px` / sm `5×12px` | |

### 5.3 徽章（Badge）
- AAA等级：金色渐变方形徽章，内白字
- 红/蓝/绿/橙徽章：对应功能色，浅底 + 边框 + 深色文字
- 标签 Tag：11px 小字号，灰色底，用于分类

### 5.4 表格（Table）
- 表头：米白背景 + 深蓝文字 + 加粗
- 行分隔：`1px solid #E5E1D6`
- Hover：金色透明 4% 高亮
- 单元格内边距：`10px 14px`

### 5.5 进度条
- 高度：默认 8px，小号 4px
- 填充色：`linear-gradient(90deg, #B8141A, #C9A063)`
- 圆角：4px

### 5.6 状态灯
- 8px 圆点 + 发光外阴影
- green / yellow / orange / red 四色
- 支持 `wm-pulse` 脉动动画（用于"实时"提示）

### 5.7 图标
- 基础图标：Heroicons（内联SVG，2px stroke）
- 业务图标：Emoji（🏛💰📈🏗🔗🔄🛡）作为MVP方案，后期替换为自定义SVG套装

## 六、七大智能体页面布局设计

### 6.1 统一布局骨架
所有智能体详情页采用一致的三段式结构：

```
┌──────────────────────────────────────┐
│ 顶部 Topbar + Navbar                 │
├──────────────────────────────────────┤
│ 面包屑（首页/智能体/当前模块）         │
├────────┬──────────────────────────────┤
│        │ 操作栏（搜索+面包屑+全局动作）│
│ 二级   │ 核心总览卡（大数据+总览可视化）│
│ 导航   │ 多维指标明细（4-6卡片网格）    │
│ 220px  │ 核心图表区（趋势/分布/热力）  │
│        │ AI智能体面板（解读+建议）    │
│        │ 关联信息（图谱/记录/对标）    │
├────────┴──────────────────────────────┤
│ 页脚                                  │
└──────────────────────────────────────┘
```

## 七、交互设计逻辑

### 7.1 导航与跳转
- 顶部主导航常驻，标识当前页高亮
- 左侧二级导航根据当前智能体动态生成
- 面包屑标识位置层级
- 所有跳转保留动画过渡（300ms）

### 7.2 数据操作交互
- **搜索**：统一搜索框，带图标、回车触发
- **筛选**：下拉选择 + 多选 Tag + 日期范围
- **分页**：尾部集中，Ant Design 风格
- **排序**：表格列头点击切换
- **批量操作**：复选框选中后浮出操作条
- **导出**：每个数据面板右上角提供导出入口

### 7.3 AI交互反馈
- **流式输出**：AI解读采用气泡 + 流式打字动画
- **思考过程**：ReAct模式显示"正在调用XX工具…"
- **引用来源**：底部列出所有RAG检索的文档与链接
- **工作流可视化**：Multi-Agent DAG实时标绿已完成节点
- **人工中断**：关键决策点弹窗等待人工确认

### 7.4 反馈机制
| 场景 | 反馈方式 |
|------|----------|
| 操作成功 | Toast绿色提示（3秒自动消失）|
| 警告 | Toast橙色提示 |
| 错误 | Toast红色提示 + 详细错误面板 |
| 加载 | 骨架屏 + 进度条 + 倒计时（若可预估）|
| 二次确认 | Modal弹窗，红色主按钮 |
| 实时更新 | 脉动点 + 时间戳 |

### 7.5 快捷键（高级用户）
- `Ctrl + K`：全局搜索
- `Ctrl + /`：AI助手
- `Esc`：关闭弹窗
- `G → H`：跳转首页
- `G → D`：跳转监管驾驶舱

## 八、响应式设计方案

### 8.1 PC端大屏自适应（主场景）
- 基于 Tailwind Grid + flex 混合布局
- 所有固定宽度限定为 `max-width`，避免溢出
- ECharts 图表使用 `chart.resize()` 在 window resize 时重绘
- 数据表格支持水平滚动（溢出时显示横向滚动条）

### 8.2 小屏降级策略
| 屏幕宽度 | 降级策略 |
|----------|----------|
| <1024px | 左侧导航折叠为抽屉 |
| <768px | 卡片纵向堆叠，KPI从4栏变2栏 |
| <480px | 极简模式，仅展示核心评分 |

### 8.3 打印样式
- 信用报告、风险报告等需要打印场景，提供专门的打印样式表
- 隐藏导航与交互元素
- 彩色元素保留，确保印刷可读

## 九、视觉装饰与国资文化元素

### 9.1 装饰细节
- **金色分割线**：深色背景下的品牌细节
- **印章式徽章**：AAA/AA/A 等级徽章采用方形 + 内白字（仿公章质感）
- **祥云纹理**：Banner背景淡化的SVG图案
- **回字纹**：卡片装饰性点缀（角落）
- **金红渐变竖条**：卡片标题栏左侧 3×16px 竖条

### 9.2 动效原则
- **稳重优先**：禁用夸张动画（弹跳、旋转）
- **仅使用**：fade、slide-up、scale(0.95→1)、脉动
- **时长**：200-400ms
- **缓动**：ease-out、cubic-bezier(0.16, 1, 0.3, 1)
