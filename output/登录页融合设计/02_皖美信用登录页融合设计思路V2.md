## 模块一 · 设计核心定位与融合原则

### 1.1 一句话设计主张
> **"长明不熄、照见诚信"——以徽州书斋一盏「皖韵长明灯」为主角，让用户拉链启灯的一刻，看着暖金光照亮书案、浮现皖美信用印鉴、铺展登录文书。**

### 1.2 为什么选"徽州书斋铜台灯"而不是普通台灯
| 维度 | 徽州书斋铜灯的适配性 |
|---|---|
| 国资合规 | 徽商故居书斋铜灯气质稳重、文人风骨，零娱乐感 |
| 信用隐喻 | "长明灯"在中国文化中代表"诚信承诺 · 日久不熄"，天然契合信用主题 |
| 双状态天然 | 熄 / 明 是最经典、最易理解的二元状态 |
| 地域识别 | 徽派马头墙暗纹浮雕于铜柱 + 朱漆铜罩 = 不可替代的安徽气质 |
| 禁娱乐化 | 徽州文房器具（笔架、镇纸、墨盒、灯）一脉，文气内敛 |
| 与 lumina 区隔 | lumina 是欧式萌系桌灯（圆锥罩、眼睛、细拉绳）；本灯是中式徽派方罩铜链（朱罩、马头墙柱、铜链铃铛）|

### 1.3 定位三元锚
| 锚点 | 内容 |
|---|---|
| 品牌锚 | 安徽国资 · 皖美信用 · 省级权威 · 监管合规 |
| 情感锚 | 稳重而不刻板 · 庄严而有温度 · 有巧思而不花哨 · 有文气而不迂腐 |
| 功能锚 | 3 秒内可见登录主路径 · 5 秒内可完成标准登录 · 0 秒障碍 |

### 1.4 融合原则（FIVE-PACT 五约 · V2.0）
| 编号 | 原则 | 约束 |
|---|---|---|
| PACT-1 | **仅承继哲学，不承继形骸** | 复用 lumina 的"拉线启灯→照亮表单"哲学，禁用 lumina 的具象（圆锥灯罩/眼睛/细拉绳/白色身）|
| PACT-2 | **国资为体，巧思为用** | 巧思服务国资品牌表达，巧思绝不凌驾于权威性之上 |
| PACT-3 | **徽派文房一以贯之** | 灯体、装饰、拉链、底座、光辉均带徽派/文房气质，禁混搭欧式 |
| PACT-4 | **色彩封域** | 严格在 WM-01~05 红金五色体系内，禁彩虹漂移 |
| PACT-5 | **功能优先级永高于动效** | 动效、音效、光影与核心功能冲突时，先砍动效 |

### 1.5 如何平衡"专业性 × 趣味性"
- **仪式感而非游戏感**：拉铜链启长明灯是文人日常动作，庄重而优雅，不是儿童触屏游戏
- **可跳过的巧思而非必玩关卡**：顶部"直接登录"Chip + 键盘 Enter 等价路径随时可用
- **一次性惊喜而非反复打扰**：首次访问触发完整仪式，sessionStorage 记忆后直接呈表单

## 模块二 · 核心交互元素与完整交互闭环设计

### 2.1 核心交互元素：**皖韵·长明灯**（原创重构）

#### 2.1.1 元素定义（原创形制）
```
名  称：皖韵·长明灯（简称"长明灯"）
灯  罩：朱漆铜方罩（不是圆锥！方形四棱罩，类徽州宫灯方罩）
       罩顶饰鎏金檐角（抽象徽派马头墙造型），罩身浅浮雕缠枝回纹
       熄灭时罩色深沉近黑红，长明时朱罩透出鎏金暖光
灯  柱：鎏金铜柱（粗壮挺拔，不是 lumina 的细白颈）
       柱身暗雕马头墙阶梯暗纹（浅浮雕）+ 回纹箍环两道
灯  芯：罩内一盏朱金双层玉髓灯芯（熄时冷灰，明时鎏金）
拉  链：黄铜小链（非 lumina 的细白绳！），长约灯柱 2/3
       链末端系一枚缠红绳的小铜铃（非环扣！铃铛带"皖"字阴文）
       铃铛在闲置时极弱摆动（幅度 ≤ 1px · 12s 周期）
灯  座：徽墨石方座（不是 lumina 的梯形黑底！），四角抹角
       座面刻极细"诚信"阳文篆字（仅启灯后桌面微光下隐约可见）
桌  面：灯座下为整张徽墨漆桌面，长明时照亮成暖金桌案，
       照映出一枚半透的"皖美信用"印鉴水痕（桌面浮痕，非空中浮印）
站  位：登录页左侧视觉中心（与右侧功能区对称均衡）
原创声明：
  · 灯罩：方罩 ≠ lumina 的圆锥罩
  · 灯身：鎏金铜柱 + 马头墙暗纹 ≠ lumina 的白色细颈
  · 拉线：黄铜链 + 铃铛 ≠ lumina 的白色细绳 + 环扣
  · 气质：徽州书斋文房 ≠ lumina 的欧式卡通萌系
  · 灯芯/辉光：朱金双层玉髓 ≠ lumina 的随机彩虹 hue
  · 底座：徽墨石方座 + 诚信阳文 ≠ lumina 的梯形普通底
  · 桌面：徽墨漆桌 + 浮印水痕 ≠ lumina 的纯色背景
  · 绝不复用：灯罩下的"眼睛"（lumina 萌系标志，本方案零萌系）
```

#### 2.1.2 元素的双状态定义
| 状态 | 命名 | 视觉形态 | 语义 |
|---|---|---|---|
| A · 熄灭态（默认） | **"长明未启"** | 朱罩深沉近黑红、铜柱黯淡、铜链铃铛轻摆、桌面徽墨如夜 | 信用未启、登录未动 |
| B · 长明态（激活） | **"长明照案"** | 朱罩透出鎏金暖光、铜柱反出金边、桌面照亮、"皖美信用"印鉴浮痕显影 | 信用长明、登录就绪 |

### 2.2 主题化交互闭环（完整用户路径）

#### 2.2.1 全流程路径图
```
[段0·进场]  页面加载 → 徽墨底渐显 → 长明灯居左呈熄灭态 → 右侧"待启案"占位
                                │
                                ▼
[段1·邀约]  0.8s 后，铜链末端铃铛处出现极弱"一次性金色呼吸光"
            配合微文字提示："拉 · 动 · 铜 · 链 · 以 · 启 · 长 · 明"
            （仅 2 秒显现后淡出）
                                │
                                ▼
[段2·操作]  用户鼠标/指针移至铜铃 → 光标变"抓取态"
            用户向下拉动铜链（可拖拽，支持键盘 Enter 等价触发）
            过程中：
              · 铜链随拖拽幅度平滑延展，模拟真实金属链的轻微晃动
              · 铃铛小幅旋摆（幅度 ≤ 6°）
              · 灯柱同步微抖（幅度 ≤ 1px），表达"临近启灯"的临界感
              · 伴随极轻"铜链摩擦 + 铃铛铃音"（低频金属共振 ≤ 120ms，间歇）
                                │
                                ▼
[段3·判定]  拖拽位移 ≥ 阈值 T（相对灯柱高度的 40%）时判定为有效启灯
            未达阈值释放：铜链以"金属链阻尼回弹"归位
                         （三段式：缓出 → 小过冲 → 稳定，参考真实链条惯性）
            达到阈值释放：进入段4
                                │
                                ▼
[段4·启灯]  （严格"起承顿收"四拍，总时长约 1.0s）
            起 · 灯芯苏醒（0.3s）：灯芯由冷灰渐变为鎏金，罩内透出第一束光
            承 · 朱罩透光（0.3s）：朱漆罩由深沉近黑渐变为朱红通透，
                               罩内鎏金灯芯亮度达峰
            顿 · 长明喷薄（0.1s）：朱罩周围一次性"鎏金光晕"扩散（≤ 240px 半径）
                               配"嗡"的一声低频长明音（≤ 6dB）
            收 · 桌面映照（0.3s）：灯光照亮桌面，"皖美信用"印鉴水痕在桌面徐徐浮现
                               （不是空中浮印！是桌面漆层下的水痕显影）
                                │
                                ▼
[段5·展卷]  右侧登录表单自"卷轴展开"呈现（宽度 0→100%，0.5s 缓出）
            表单各字段从上至下错峰进场（每字段 60ms stagger）
            卷轴顶底饰鎏金轴杆，卷轴底色为浅徽宣
                                │
                                ▼
[段6·长明呼吸]  全页面进入"长明呼吸"：
              · 背景徽墨 → 徽墨带极弱朱金呼吸光（周期 8s · 幅度 ≤ 6%）
              · 灯罩本身保持稳定长明，不再波动（长明的承诺感）
              · 桌面浮印水痕 0% → 4% 透明度微微显影
              · 铃铛恢复极弱闲置摆动
                                │
                                ▼
[回退]    用户点击表单右上角"熄灯"小图标，反向触发：
          表单卷轴回卷 → 桌面印鉴水痕淡出 → 朱罩渐暗 → 灯芯熄灭 → 
          铜链铃铛回归闲置 → 背景呼吸光熄灭
          极轻"铃铛轻响 · 灯芯熄灭"声（≤ 80ms）
```

#### 2.2.2 每一步的操作反馈规则
| 步骤 | 视觉 | 动效 | 音效 | 触觉（预留） |
|---|---|---|---|---|
| 段1 邀约 | 铃铛金色呼吸光 2s 淡入淡出 | 无 | 无 | 无 |
| 段2 操作 | 铜链延展 + 铃铛旋摆 + 灯柱微抖 | 1:1 位移跟随 | 铜链摩擦 + 铃音 ≤ 120ms | 可选振动 |
| 段3 回弹 | 铜链三段阻尼回弹 | 金属链惯性曲线 | 铃铛轻响 | 无 |
| 段4 启灯 | 灯芯苏醒→朱罩透光→长明喷薄→桌面映照 | 严格起承顿收四拍 | 单次低频"嗡"长明音 ≤ 6dB | 一次短振 |
| 段5 展卷 | 卷轴展开 + 字段错峰 | 缓出 + 60ms stagger | 极轻纸卷展开声 ≤ 80ms | 无 |
| 段6 长明呼吸 | 全页面徽墨朱金呼吸 | 正弦 8s | 无 | 无 |
| 回退 熄灯 | 全部反向 | 反向曲线 | 铃铛轻响 + 熄灭声 ≤ 80ms | 无 |

#### 2.2.3 判定条件与可控性底线
- 拖拽阈值 T 默认 40% 灯柱高度，可在用户研究后标定
- 若用户 0.5s 内连续触发 3 次未达阈值释放，自动弹出"改用直接登录"引导
- 整个交互 2s 无响应则判定"环境异常"，降级为静态登录页

### 2.3 三条可替代主路径
| 通道 | 触发条件 | 路径 |
|---|---|---|
| **主通道·仪式启灯** | 首访 + 鼠标用户 | 上文段 0–6 |
| **快捷通道·键盘启灯** | Tab 聚焦到铃铛 + Enter/Space | 跳过段 2 过程动画，直接走段 4–6 |
| **无障碍通道·直接登录** | 屏读用户 / 已访问过 / 点"直接登录"Chip | 跳过段 0–5，直接呈现静态表单 |

## 模块三 · 视觉与动画设计规范（双版本）

### 3.1 色彩封域 · 皖美红金五色体系（跨版本共享）

| 色号 | 色名 | 色彩角色 | 典型用途 |
|---|---|---|---|
| WM-01 | **徽墨玄** | 沉稳底色 | 深色版背景、桌面、文字 |
| WM-02 | **皖白宣** | 克制浅底 | 浅色版背景、表单底 |
| WM-03 | **朱泥红** | 核心主色 | **灯罩朱漆**、主 CTA、合规强调 |
| WM-04 | **鎏金晖** | 权威高光 | **灯芯辉光**、铜柱鎏金、LOGO 辅色 |
| WM-05 | **黛青线** | 辅助线条 | 分隔线、装饰回纹、次文本 |

**封域规则**：页面动态变色必须在 WM-03/WM-04 之间微调，禁止彩虹漂移。

### 3.2 版本一 · 深色沉浸专业版（会议室/大屏）

#### 3.2.1 配色基调
- 背景主色：徽墨玄（WM-01）
- 装饰暗纹：徽派马头墙远山 + 书案阴影，透明度 ≤ 4%
- 灯罩主色：朱泥红（WM-03），熄时朱红压暗 50% 近黑红
- 灯芯/辉光：鎏金晖（WM-04），仅启灯后呈现
- 桌面：灯座下徽墨漆（熄时）→ 暖金反光（明时）
- 整体对比度 ≥ 7:1

#### 3.2.2 布局规范
- 主栅格：12 列，中线居中
- 长明灯居左 5 列，登录面板居右 5 列，两侧各 1 列留白
- 顶部 80px 锚定：LOGO + 快捷通道
- 底部 64px 锚定：合规 + 版权

#### 3.2.3 动画节奏（中式起承顿收）
| 阶段 | 曲线 | 持续 |
|---|---|---|
| 起（苏醒） | easeOutCubic / power2.out | 0.3s |
| 承（透光） | easeInOutQuart / power3.inOut | 0.3s |
| 顿（喷薄） | back.out(0.8) 极弱过冲 | 0.1s |
| 收（照案） | easeOutSine | 0.3s |

**严禁** elastic.out / bounce.* 这类西式强弹性。

#### 3.2.4 光影联动逻辑
- 段6 全局呼吸光：周期 8s · 幅度 ≤ 6% · 不抢注意力
- 段4 启灯瞬间：朱罩外一次"鎏金光晕扩散波"（≤ 240px 半径 · 0.3→0 · 0.6s）
- 桌面反光：用 CSS `radial-gradient` + `mix-blend-mode: screen` 模拟暖金照射

### 3.3 版本二 · 浅色稳重办公版（日常办公）

#### 3.3.1 配色基调
- 背景主色：皖白宣（WM-02）
- 装饰暗纹：宣纸纤维 + 徽派线描远山，透明度 ≤ 8%
- 灯罩主色：朱泥红，饱和度较深色版降低 10%
- 灯芯/辉光：淡金晕（WM-04），透明度上限 12%
- 桌面：浅木案色（启灯时暖金反光更柔和）
- 对比度 ≥ 4.5:1（WCAG AA）

#### 3.3.2 关键差异
| 维度 | 深色沉浸版 | 浅色稳重版 |
|---|---|---|
| 光影手段 | 鎏金辉光 + 暖金扩散波 | 淡金晕染 + 宣纸吸光 |
| 灯罩质感 | 朱漆 + 鎏金高光边 | 朱漆 + 浅金描边 + 宣纸透光肌理 |
| 启灯扩散 | 鎏金扩散波 | 暖金墨晕（宣纸吸墨感）|
| 桌面反光 | 强暖金 screen 叠加 | 柔浅金 multiply 叠加 |
| 音效音量 | ≤ 6dB | ≤ 4dB |

#### 3.3.3 版本切换逻辑
- URL 参数：`?mode=dark|light`
- 页面右上角"皖墨/皖白"切换键
- localStorage 记忆
- 默认深色（国资会议室/大屏优先）

### 3.4 CSS 设计令牌绑定规则
- 根级 `data-theme="dark|light"` + `data-stage="DIMMED|LIGHTING|BRIGHT|DIMMING"`
- 所有色值、透明度、阴影、模糊半径抽象为令牌
- 状态切换通过根属性驱动，所有组件订阅

### 3.5 动画与交互状态的强绑定规则
| 状态 | 绑定动画 | 可中断 |
|---|---|---|
| 熄灭态静置 | 铃铛极弱摆动（幅度 ≤ 1px · 12s） | 可 |
| 铜链拖拽中 | 1:1 位移跟随 + 铃铛旋摆 | 不可 |
| 释放未达阈值 | 三段阻尼回弹 | 不可 |
| 启灯四拍 | 起承顿收 | 不可（保护仪式感）|
| 卷轴展开 | 宽度 + 字段 stagger | 可（提前 Enter 立即完成）|
| 长明呼吸 | 正弦 8s | 可（任何交互暂停）|

## 模块四 · 全量功能融入设计（100% 覆盖，易用性零损）

### 4.1 基础登录功能

#### 4.1.1 账号输入框
- 位于表单第一行，宽度 100%
- 弱化边框 + 下方 1px 黛青线 + 左侧"国密账号"徽纹图标
- 聚焦时下方线条由黛青变朱泥红，宽度由短延伸全宽（0.3s）
- 占位："请输入账号 / 手机号 / 工号"

## 模块五 · 原创性差异化说明（相对 lumina-login）

### 5.1 全量差异对照表（11 维完全不同）
| 维度 | lumina-login | 本方案 V2.0 | 差异评级 |
|---|---|---|---|
| 灯罩形状 | 圆锥/半圆白罩 | **徽派方罩（朱漆+鎏金檐角）** | ★★★★★ |
| 灯身材质 | 细白颈 + 梯形底 | **鎏金铜柱 + 马头墙暗纹 + 徽墨石方座** | ★★★★★ |
| 拉动机构 | 白色细绳 + 环形扣 | **黄铜链 + 缠红绳铜铃铛（"皖"字阴文）** | ★★★★★ |
| 装饰语言 | 卡通萌系 + 眼睛 + 瞳孔反光 | **徽派回纹 + 马头墙浮雕 + 诚信阳文** | ★★★★★ |
| 色彩体系 | 随机 hue 0–360 彩虹漂移 | **WM 红金封闭五色 + 朱金二色辉光域** | ★★★★★ |
| 主题隐喻 | "开灯→照亮表单" | **"拉铜链→启长明→照案浮印→展卷登录"** | ★★★★★ |
| 动画哲学 | 西式 elastic.out 弹性回弹 | **中式"起承顿收"四拍节奏 + 金属链阻尼** | ★★★★★ |
| 音效设计 | 拉绳沙沙 + 咔哒金属 | **铜链摩擦 + 铃铛铃音 + 长明嗡鸣** | ★★★★★ |
| 背景装饰 | 抽象彩光光斑 | **徽派远山 + 书案漆面 + 印鉴水痕浮显** | ★★★★★ |
| 版权署名 | "Designed by Lumina Studio" | **安徽省国资委主管 + 皖 ICP 备** | ★★★★★ |
| 萌系表达 | 台灯眼睛（核心萌点） | **无（严禁 · 文房器具不可萌化）** | ★★★★★ |
| 双状态语义 | 关灯/开灯（家居日常）| **熄灭/长明（文人诚信承诺）** | ★★★★★ |

### 5.2 唯一复用点：一条通用交互哲学
> "用户主动可控操作 → 触发二元状态切换 → 核心功能区联动揭示 → 全页面视觉呼应"

此为通用交互设计方法论（类比"所有下拉刷新都共享下拉→释放→刷新"哲学），不构成具象复用。

### 5.3 V2.0 相对 V1.0 的独立原创声明
- **"皖韵·长明灯"**：本次方案独立创造的复合元素名，结合"皖韵"（徽州韵味）+"长明灯"（诚信隐喻）
- **"徽派方罩铜灯"**：朱漆方罩 + 鎏金檐角 + 铜柱马头墙暗纹 + 铜链铃铛 的完整组合为本次原创组合
- **"长明照案"命名体系**：长明未启 / 长明照案 / 灯下留名 / 核验长明 / 熄灯归寂 为本次原创
- **"起承顿收"四拍**：中式动画哲学沿用 V1.0 提炼，与任何参考样本动画曲线无关
- 版权归安徽省国资委及承建单位所有
