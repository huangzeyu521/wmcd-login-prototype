---

## 一、技术选型

| 层 | 选型 | 版本 | 理由 |
|---|---|---|---|
| 页面架构 | 原生 HTML5 + CSS3 + ES2017+ JS | — | 匹配现有平台（静态页面架构），零打包 |
| 样式框架 | Tailwind CDN + 自研 shared.css + login-wmcd.css | Tailwind 3 | 与平台一致，新增页专属 CSS |
| 动画引擎 | **GSAP 3.12 (CDN)** + CSS Transitions | 3.12.5 | 满足设计方案"GSAP 等动画库"硬性要求；复杂编排用 GSAP，简单用 CSS |
| 拖拽 | **原生 Pointer Events**（不引入 GSAP Draggable） | — | 零依赖 + 符合"避免过度依赖"建议 |
| SVG | 原生 inline SVG | — | 可审计、可被 CSS 直接驱动 |
| 音频 | **Web Audio API**（自合成） | — | 0KB 资源 + 音色完全原创（与 lumina 的拉绳音完全不同） |
| 状态机 | 原生 JS FSM（无外部库） | — | 简单需求不引 XState |
| 模拟登录 | 复用 `WM_AUTH` / `WM_USERS` / `WM_STORE` | 现有 | 平台依赖，不可破坏 |
| 字体 | 思源宋/思源黑（system fallback） | — | 无境外依赖 |

## 三、开发阶段拆分

### P0 · 初始化与基础架构（估 20 分钟）
**任务**
1. 备份现有 `login.html` → `login-legacy.html`
2. 创建 `login-wmcd.css` 骨架 + WM 五色令牌（深/浅双版本）
3. 创建 `wm-seal-svg.js` 空模块
4. 创建 `login-wmcd.js` 空模块
5. 新建 `login.html` 顶层骨架（DOCTYPE + viewport + 主要容器）

**交付物**：基础架构可访问（浏览器打开不报错）
**完成标准**：浏览器打开 `http://localhost:8765/pages/login.html` 显示空骨架无控制台报错

---

### P1 · 核心交互元素：皖美信用之印 SVG（估 40 分钟）
**任务**
1. 绘制印盒（底座 + 内腔 + 淡金描边）
2. 绘制朱漆绶带（两条飘带 + 系扣）
3. 绘制印章身（方印体 + 马头墙印钮 + 朱泥正面）
4. 绘制印面（篆隶"皖美信用"四字）
5. 绘制落印印鉴（page-center 版本）
6. 定义 SEALED / UNSEALED 双态静态 CSS

**交付物**：`wm-seal-svg.js` 导出 `buildSealSVG()` 方法 + 静态双态渲染
**完成标准**：浏览器刷新时，默认看到完整封存态信印；通过 DevTools 改 `data-stage="UNSEALED"` 能看到启封态

---

### P2 · 核心交互逻辑（估 60 分钟）
**任务**
1. 实现有限状态机 FSM（SEALED ↔ RELEASING ↔ UNSEALED ↔ RESEALING）
2. 实现绶带 Pointer 拖拽（1:1 位移跟随 + 阈值 40% 判定）
3. 实现三段阻尼回弹（未达阈值）
4. 实现起承顿收四拍（达阈值）
5. 实现落印印鉴浮现 + 卷轴展开 + 字段 stagger
6. 实现全局呼吸光（正弦 8s, 幅度 ≤ 6%）
7. 实现回退"归印"动画
8. 实现 Web Audio 音效：绶带丝声（高通白噪 ≤ 120ms）+ 落印嗡音（低频正弦 0.6s 包络 ≤ 6dB）+ 纸展声 + 收印声
9. 键盘等价路径：Tab + Enter/Space 触发

**交付物**：完整交互闭环
**完成标准**：拖拽/键盘均可达成 SEALED→UNSEALED / UNSEALED→SEALED 的切换，动画流畅，无控制台报错，音效正常

---

### P3 · 双版本视觉样式（估 40 分钟）
**任务**
1. 深色沉浸版：徽墨底 + 远山暗纹 + 辉光启封
2. 浅色稳重版：皖白底 + 宣纸纤维 + 墨晕启封
3. 版本切换按钮（`data-theme` 根属性）+ localStorage 记忆
4. `?mode=dark|light` URL 参数支持
5. `prefers-color-scheme` 系统默认跟随
6. 对比度校验（深色 7:1 / 浅色 4.5:1）

**交付物**：双版本可一键切换
**完成标准**：切换按钮点击、URL 参数、系统偏好三路径均可触达双版本，视觉还原符合设计

---

### P4 · 全量表单与国资专属功能（估 50 分钟）
**任务**
1. 顶部 80px 锚定：LOGO + 皖美信用文字标识 + 英文 + 右侧快捷入口（SSO/CA/U-Key/版本切换/直接登录）
2. 表单区（卷轴容器内）：账号 + 密码 + 记住密码 + 忘密 + 登录按钮
3. 密码显隐切换（封/启徽纹图标）
4. 大写锁定检测
5. 记住密码（默认 false + 仅记账号）
6. 忘密合规浮层
7. 官方认证通道区块（政务 SSO + CA + U-Key 三按钮）
8. CA 驱动检测 5s 超时降级卡片
9. 底部合规 + 署名
10. 兼容现有角色选择（作为"或选择角色快速体验"的 MVP 入口，折叠在登录按钮下方）

**交付物**：表单全功能可用
**完成标准**：所有字段正常输入，登录按钮可通过账号+密码 OR 角色选择两种方式登录，跳转 workbench.html

---

### P5 · 响应式适配与无障碍（估 30 分钟）
**任务**
1. 屏幕尺寸断点：1920+ / 1440-1920 / 1280-1440 / 1024-1280 / <1024
2. 信印尺寸与表单宽度按断点映射
3. <1024 上下堆叠降级
4. 所有图形 `aria-label` / `role`
5. Tab 键序：先表单再信印（老用户优先）
6. 键盘 Enter/Space 触发启封
7. `prefers-reduced-motion` 自动静态降级
8. 高对比度模式（`@media (prefers-contrast: more)`）

**交付物**：5 档尺寸 + 无障碍完整
**完成标准**：Chrome DevTools 切换不同尺寸无错乱；键盘操作可完成完整登录；屏读器可读

---

### P6 · 兼容性加固（估 20 分钟）
**任务**
1. GSAP 加载失败时降级为 CSS 动画
2. Web Audio 不支持时静默跳过音效
3. Pointer Events 不支持时降级为 Mouse Events
4. SVG 不支持时显示静态图片占位
5. IE 11 完全降级为 `login-legacy.html`
6. 低速场景首屏骨架优先

**交付物**：全浏览器 fallback 完整
**完成标准**：断网 / 禁 JS / 禁 SVG / 禁 Audio 多组异常场景下登录功能依然可用

---

### P7 · 全维度测试与迭代（估 30 分钟）
**任务**
1. 功能测试（输入/提交/跳转）
2. 交互测试（拖拽/键盘/回退）
3. 视觉测试（双版本还原）
4. 响应式测试（5 档尺寸）
5. 兼容性测试（Chrome/Edge/Firefox）
6. 性能测试（帧率/包体）
7. 合规测试（品牌调性/红线）
8. 输出《BUG 修复清单》+《全量测试报告》

**交付物**：测试报告 + BUG 清单（清零）
**完成标准**：所有测试项 100% 通过，零 BUG

## 六、代码规范

### 6.1 结构
- HTML：语义化标签 + data-* 状态驱动
- CSS：BEM 命名 + `--wm-*` 令牌 + `data-theme` / `data-stage` 根控
- JS：IIFE 模块 + 单一职责 + JSDoc 注释关键函数

### 6.2 命名
- CSS 类：`wm-seal-*` / `wm-login-*` / `wm-scroll-*`（本页专属）
- JS 函数：`camelCase`
- 常量：`UPPER_SNAKE_CASE`
- FSM 状态：`SEALED / RELEASING / UNSEALED / RESEALING`（枚举）

### 6.3 注释
- 每个 JS 文件顶部：功能说明 + 依赖
- 关键函数：JSDoc 简注释
- 复杂动画序列：时序说明

### 6.4 性能
- 禁用 `console.log` 在生产路径
- 动画优先 `transform` / `opacity`（compositor 友好）
- 避免大量重排（绶带 path 更新用 requestAnimationFrame throttle）

## 七、风险与对策

| 风险 | 对策 |
|---|---|
| GSAP CDN 不可达 | `window.gsap` 存在性检测 + CSS keyframes 降级 |
| 现有 `auth.js` 依赖破坏 | 保留现有 `WM_AUTH.login(userId)` 调用接口，新增表单登录最终也调此方法 |
| 信印 SVG 渲染性能 | path 数量控制在 30 以内，避免大量 filter |
| 音效打扰用户 | 首次需用户点击触发（浏览器 autoplay policy），可关闭 |
| 键盘与拖拽冲突 | FSM 严格互斥 + pointer-events 动态控制 |
