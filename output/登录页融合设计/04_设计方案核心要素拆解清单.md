## 一、核心交互元素拆解

### 1.1 皖美信用之印（信印）组成要素
| 层级 | 构件 | 设计描述 | 开发落点 |
|---|---|---|---|
| L1 | 印盒（底座） | 徽墨色扁矩形底座 + 淡金描边 | SVG `<rect>` + 双 stroke |
| L2 | 印盒内腔 | 朱红丝绒纹理 | SVG 渐变填充 |
| L3 | 朱漆绶带 | 两条朱红飘带 + 丝绸纹 + 金线 | SVG `<path>`（可形变） |
| L4 | 绶带系扣 | 金色环扣（拖拽把手） | SVG `<circle>`（可拖拽） |
| L5 | 印章身（方印体） | 朱泥方体 + 徽派马头墙钮顶 | SVG `<path>` 组合 |
| L6 | 印钮（马头墙） | 徽派马头墙抽象几何（非写实） | SVG `<polygon>` 阶梯 |
| L7 | 印面（朝下/朝外） | 篆隶"皖美信用"四字 + 朱砂底 | SVG `<text>` + 方框 |
| L8 | 落印印鉴 | 1:1 真实"皖美信用"印记 | SVG + CSS mask |
| L9 | 光晕 | 鎏金辉光（启封后）/ 墨晕（浅色版） | SVG `<circle>` + filter blur |

### 1.2 双状态定义（状态机元数据）
| 状态 ID | 中文 | 视觉形态 | 页面表现 |
|---|---|---|---|
| SEALED | 信印封存（默认） | 印身嵌入印盒，绶带系扣朝上，印面朝下 | 右侧呈"待启封"占位，仅静态登录入口 |
| RELEASING | 启封中（瞬态） | 信印上升+翻转中 | 锁定输入，禁止重复触发 |
| UNSEALED | 信印已启 | 印身悬浮于印盒上方 20px，印面朝外 | 右侧表单卷轴展开，页面呼吸光开启 |
| RESEALING | 归印中（瞬态） | 反向动画 | 表单卷回，信印回落 |

### 1.3 交互闭环时序矩阵
| 段 | 触发 | 时长 | 动画 | 音效 | CSS 令牌变化 |
|---|---|---|---|---|---|
| 段0 进场 | 页面 DOM ready | 0.6s | 徽墨渐显 + 印盒呼吸启动 | 无 | `--stage=entering` |
| 段1 邀约 | 0.8s 后 | 2s | 绶带系扣金色呼吸光 + 微文字淡入 | 无 | `--invite=on` |
| 段2 操作 | pointerdown on handle | 持续 | 绶带 1:1 跟随 + 印盒微抖（max 2px） | 绶带丝声 ≤ 120ms（间歇）| 无 |
| 段3A 回弹 | pointerup < 阈值 | 0.6s | 三段阻尼（缓出→小过冲→稳定） | 无 | 无 |
| 段3B 启封 | pointerup ≥ 阈值 | 起 0.4 + 承 0.2 + 顿 0.15 + 收 0.25 = 1.0s | 起承顿收四拍 | 0.6s 内单次落印嗡音 ≤ 6dB | `--stage=unsealing` |
| 段4 落印 | 启封后 | 0.6s | 中心浮现"皖美信用"印鉴 0.3 → 0.5 → 0 | 极轻纸展声 ≤ 80ms | `--stage=stamped` |
| 段5 揭示 | 紧随段4 | 0.5s + stagger 0.08s | 卷轴 width 0→100% + 字段错峰入场 | 无 | `--form=visible` |
| 段6 照映 | 揭示完成 | 持续 | 徽墨朱金呼吸 ≤ 6% / 正弦 8s | 无 | `--state=UNSEALED` |
| 回退 归印 | 点击归印按钮 | 0.9s | 全部反向 | 极轻收印声 ≤ 80ms | `--state=SEALED` |

### 1.4 三条主路径
| 路径 | 触发 | 跳过段 | 技术实现 |
|---|---|---|---|
| P1 仪式启封 | 首访 + 鼠标用户 | 全走 | Pointer Events + GSAP |
| P2 键盘启封 | Tab 到系扣 + Enter/Space | 跳段2 | keydown 监听触发段3B |
| P3 直接登录 | 屏读 / sessionStorage 已访问 / 点"跳过仪式"链接 | 跳段0-段5 | sessionStorage + `?mode=direct` URL |

## 二、视觉设计规范拆解

### 2.1 WM 红金五色令牌
| 令牌 | 色号（深色版） | 色号（浅色版） | 用途 |
|---|---|---|---|
| `--wm-huimo` | `#0D1119` 徽墨玄 | — | 深色版主背景 / 深色文字 |
| `--wm-wanbai` | — | `#F5F1E6` 皖白宣 | 浅色版主背景 / 表单底 |
| `--wm-zhuni` | `#B8141A` 朱泥红 | `#A51319` 深版减饱和 10% | 信印印面 / 主 CTA / 强调 |
| `--wm-liujin` | `#C9A063` 鎏金晖 | `#B88E4A` 淡金晕 | 边框描金 / 高光 / LOGO 辅色 |
| `--wm-daiqing` | `#4A5563` 黛青线 | `#6A7580` 浅版 | 分隔线 / 装饰徽纹 / 次文本 |

### 2.2 版本切换矩阵
| 项目 | 深色沉浸专业版 | 浅色稳重办公版 |
|---|---|---|
| 背景 | 徽墨玄实色 + 远山渐变暗纹 | 皖白宣 + 宣纸纤维 + 徽派线描远山 |
| 主交互色 | `#B8141A` 朱泥红 | `#A51319` 减饱和朱泥红 |
| 启封光晕 | 鎏金辉光（扩散波） | 淡金墨晕（宣纸吸墨） |
| 印章材质 | 金边高光 + 辉光 | 纸贴质感 + 轻阴影 + 纤维纹 |
| 对比度 | ≥ 7:1 | ≥ 4.5:1（WCAG AA） |
| 音效音量 | ≤ 6dB | ≤ 4dB |

### 2.3 动画曲线（中式起承顿收）
| 阶段 | 持续 | cubic-bezier / GSAP ease | 备注 |
|---|---|---|---|
| 起 | 0.3-0.4s | `cubic-bezier(.25,.8,.35,1)` / `power2.out` | 缓出 |
| 承 | 0.2-0.3s | `cubic-bezier(.77,0,.18,1)` / `power3.inOut` | 平滑延展 |
| 顿 | 0.05-0.1s | `cubic-bezier(.18,1.2,.6,1)` / `back.out(0.8)` | 极弱过冲 |
| 收 | 0.15s | `cubic-bezier(.39,.58,.57,1)` / `sine.out` | 稳定收落 |

**严禁**：`elastic.out(1, 0.4)`、`bounce.*`。

### 2.4 CSS 变量绑定规则
- 根级状态：`data-theme="dark|light"` + `data-stage="SEALED|UNSEALED|..."`
- 颜色、阴影、圆角、间距均以 `--wm-*` 令牌引用，不硬编码
- 版本切换仅改 `data-theme`，所有组件自动响应

## 四、响应式与兼容性要求

### 4.1 屏幕适配（锁死）
| 尺寸 | 信印 SVG | 表单宽度 | 行为 |
|---|---|---|---|
| ≥ 1920 | 480px | 480px | 完整体验 + 大留白 |
| 1440-1920 | 420px | 440px | 完整体验 |
| 1280-1440 | 360px | 400px | 完整体验 |
| 1024-1280 | 300px | 360px | 完整体验（默认优先面） |
| < 1024 | 240px | 100% | 上下堆叠降级 |

### 4.2 浏览器兼容
- 完整体验：Chrome/Edge/Firefox（最近 3 版）、360、国产浏览器主流版
- 降级静态：IE 11（禁用 GSAP + 禁用信印 + 仅静态表单 + SSO 优先）

### 4.3 性能预算
- 首屏 FCP ≤ 1.5s / TTI ≤ 2.5s
- 信印 SVG ≤ 80KB / 字体子集 ≤ 120KB / 总下载 ≤ 400KB
- 动画帧率 ≥ 55fps

## 五、合规要求

| 项 | 要求 | 落点 |
